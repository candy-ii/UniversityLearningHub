# UniversityLearningHub
A platform for students to share resources and communicate with each other.

Setup Guide:

→ Install Node.js (LTS) and Git.

→ Go to Source Control on the left

→ Clone Repository

→ Clone from Github

→ Paste https://github.com/candy-ii/UniversityLearningHub.git

→ Select a folder

→ Login to github at the top

→ At the bottom left, you will see something called 'main'. Click it and select the branch with your name.

Now, install necessary packages with the following commands:

> cd server
> npm install

> cd client
> npm install

You will need also need to create a .env file in /server, then you're set!
